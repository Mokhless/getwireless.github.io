Ec/No

Eb: Bit Energy. Il représente la quantité d'énergie par bit.

No: Noise Spectral Density. Unit: Watts/Hz (or mWatts/Hz)

Ec/No Range (dB)
Ec/No_00 Eclo < -24
Ec/No_01 -24 <= Ecl0 < -23.5
Ec/No_02 -23.5 <= Eclo < -22.5
... ...
Ec/No_47 -1 <= Eclo < -0.5
Ec/No_48 -0.5 <= Eclo < 0
Ec/No_49 0 <= Ecl0
