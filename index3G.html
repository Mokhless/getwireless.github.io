RSCP

Received Signal Code Power est la puissance mesurée par un récepteur sur un canal de communication physique particulier

Range (dBm) Quality Color
RSCP < -62 Very Good Coverage
-83 < RSCP < -77 Good Coverage
-95 < RSCP < -83 Poor Coverage
-104 < RSCP < -95 Bad Coverage
-104 < RSCP No Coverage
Ec/No

Eb: Bit Energy. Il représente la quantité d'énergie par bit.

No: Noise Spectral Density. Unit: Watts/Hz (or mWatts/Hz)

Range (dB) Quality Color
Ec/No < -5 very Good
-5 < Ec/No < -8 Good
- 8 < Ec/No < -12 Poor
-12 < Ec/No < -14 Bad
-14 < Ec/No < -16 Bad
-16 < Ec/No < -18 Bad
Ec/No < -18 Very Bad
